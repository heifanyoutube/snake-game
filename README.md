# Classic Snake Game (貪食蛇) 🐍

這是一個使用 HTML5 Canvas 和 JavaScript 開發的經典貪食蛇遊戲。
支援電腦鍵盤操作，也支援手機觸控滑動！

## 🎮 遊戲試玩 (Demo)
[點擊這裡開始玩！](https://heifanyoutube.github.io/snake-game/)
*(請將連結中的 `你的GitHub帳號` 替換成你真實的帳號)*

## ✨ 特色
- **跨平台支援**：電腦用鍵盤，手機用滑動手勢。
- **響應式設計**：自動適應螢幕大小。
- **純生代碼**：無任何外部依賴庫，輕量快速。

## 🕹️ 操作方式
| 平台 | 操作 |
| --- | --- |
| **電腦** | 使用方向鍵 `↑` `↓` `←` `→` 控制移動 |
| **手機** | 在遊戲區域內 `上下左右` 滑動手指 |

## 🛠️ 技術棧
- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 如何在本地運行
1. 下載此專案。
2. 直接用瀏覽器打開 `index.html` 即可遊玩。
